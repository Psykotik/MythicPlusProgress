# MythicPlusProgress

Addon WoW (Retail) qui affiche, en texte, tes meilleurs runs Mythic+ de la saison (score total + liste des donjons), ainsi que ta clé actuelle.

## Commandes

- `/mpp` : rafraîchir l’affichage
- `/mpp keystone` : afficher/masquer “My Keystone”
- `/mpp pve` : afficher uniquement quand la fenêtre PVE est ouverte
- `/mpp unlock` / `/mpp lock` : déverrouiller/verrouiller (drag)
- `/mpp reset` : réinitialiser la position

Alias : `/mythicplusprogress` fonctionne aussi.
